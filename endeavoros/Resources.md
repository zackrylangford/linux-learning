# 📚 EndeavourOS & Arch Resource Hub

Absolutely—that’s a smart move. Here’s a curated list of essential EndeavourOS and Arch-based reading resources. These are ideal for regular check-ins or deeper dives and will keep you sharp as both an EndeavourOS user and an Arch-level learner.

Save this as a local markdown or plain text file (e.g., `~/Documents/endeavouros-resources.txt`).

---

## 🛰️ Official Docs & Project Sites

- **[EndeavourOS Homepage](https://endeavouros.com)**
- **[EndeavourOS Discovery Wiki](https://discovery.endeavouros.com)** → Beginner friendly; great for hardware, theming, AUR, kernel tips
- **[EndeavourOS GitHub](https://github.com/endeavouros-team)** → Explore system tools like `eos-welcome`, `eos-log-tool`
- **[EndeavourOS Forum](https://forum.endeavouros.com)** → Excellent for troubleshooting, system errors, AUR issues
- **[EndeavourOS News Blog](https://endeavouros.com/news/)** → Use for weekly reading habit or Newsboat

---

## ⚙️ Arch Linux Power Resources

- **[Arch Wiki](https://wiki.archlinux.org)** → Your ultimate manual; must-read sections: `pacman`, `systemd`, `journalctl`, `PKGBUILD`, `makepkg`
- **[Arch Linux Homepage](https://archlinux.org)**
- **[Arch Package Search](https://archlinux.org/packages)** → Quickly look up what repo a package lives in
- **[Arch User Repository (AUR)](https://aur.archlinux.org)** → Search user-submitted packages; useful with `yay`

## 💡 Bonus Tools

- `eos-welcome` → Launcher for config tools, mirrors, help
- `man pacman` → Local reference manual
- `eos-update-notifier` → Set to notify you of system updates
- `reflector-simple` → GUI mirrorlist refresher
